PQC ALGORITHM RECOMMENDATIONS BASED ON PCA ANALYSIS
======================================================================

HIGH PERFORMANCE APPLICATIONS
-----------------------------
Description: Applications requiring maximum speed
PCA Focus: Low PC1 values (performance dimension)
Characteristics: Fast key generation, low encryption/decryption times
Recommended Algorithms:
  • Classic-McEliece-8192128
  • Classic-McEliece-6960119
  • Classic-McEliece-6688128
  • Classic-McEliece-8192128f
  • Classic-McEliece-6688128f

MEMORY CONSTRAINED ENVIRONMENTS
-------------------------------
Description: IoT devices, embedded systems with limited memory
PCA Focus: Consider ciphertext_length and overhead_bytes contributions
Characteristics: Small key sizes, minimal memory footprint
Recommended Algorithms:
  • Classic-McEliece-348864
  • Classic-McEliece-348864f
  • Classic-McEliece-460896
  • Classic-McEliece-460896f
  • Classic-McEliece-6960119

HIGH SECURITY REQUIREMENTS
--------------------------
Description: Applications requiring maximum security (256-bit)
PCA Focus: Higher security level clustering in PCA space
Characteristics: Larger keys, stronger security guarantees
Recommended Algorithms:
  • Classic-McEliece-8192128
  • Classic-McEliece-6960119
  • Classic-McEliece-6688128
  • Classic-McEliece-8192128f
  • Classic-McEliece-6688128f

BALANCED APPLICATIONS
---------------------
Description: General purpose applications needing balance
PCA Focus: Moderate values across all PCs
Characteristics: Good performance with adequate security
Recommended Algorithms:
  • BIKE-L3
  • BIKE-L5
  • BIKE-L1
  • Classic-McEliece-348864
  • Classic-McEliece-460896f
